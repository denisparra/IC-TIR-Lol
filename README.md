## Prerequisites

The code is built with following libraries:

- [PyTorch](https://pytorch.org/) 1.0 or higher
- [Comet-ml](https://www.comet.ml/site/)
- [PyTorchLightning](https://github.com/PyTorchLightning/pytorch-lightning)
